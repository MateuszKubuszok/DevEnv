# My command line toolset

Just my command line toolset.

## Installing

```bash
git clone git@github.com:MateuszKubuszok/DevEnv.git command-line-tools
cd command-line-tools
./install.sh
```

## Moving

```bash
cd command-line-tools
./unlink.sh
cd ..
mv command-line-tools X
cd X
./install
```
